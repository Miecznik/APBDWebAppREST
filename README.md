# APBD Szkolenia

## Opis Projektu

Web API napisane w frameworku ASP.NET mający za zadanie zarządzanie salami szkoleniowymi oraz ich rezerwacjami

## Opis dzialania

Api ma szereg endpointów które obsługują 2 główne controllery 

#### Rooms
posiada funkcje GET , GET/{id}, GET/{BuildingCode}, GET z query, POST, PUT i DELETE

#### Reservations
posiada funkcje GET , GET/{id}, GET z query, POST, PUT i DELETE
